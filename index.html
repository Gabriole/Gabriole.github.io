BMI Calculator

A Python command-line application to calculate and categorize your BMI.

About the Project

This project, BMI Calculator, was developed as part of a personal interest in understanding health indicators like Body Mass Index (BMI). It helps individuals quickly calculate their BMI by providing weight and height inputs and categorizes them into different health ranges. This project is part of my portfolio for Holberton School, and it demonstrates my ability to develop Python-based applications that are simple but impactful. You can check out more about Holberton

Key Features

BMI Calculation

The application calculates the BMI value based on user inputs for weight (in kilograms) and height (in meters).

BMI Categories

Once the BMI is calculated, the application categorizes the user into one of the following categories: Underweight, Normal weight, Overweight, or Obese.

Simple Interface

The BMI Calculator runs in the command line, making it fast, lightweight, and easy to use without the need for a graphical interface.
