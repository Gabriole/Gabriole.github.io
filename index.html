BMI Calculator

Calculate and categorize your BMI based on weight and height.

About the Project

This project was developed to help individuals quickly calculate their BMI by providing weight and height inputs and categorizes them into different health ranges.

The inspiration for this project came from my personal interest in understanding how health indicators like BMI can offer insights into one's overall health. Growing up, I witnessed many people, including family members, struggle with weight management and the associated health issues. I wanted to create something simple, yet impactful, that could help people take the first step toward understanding their health.

This project was also a great learning opportunity. Over a span of several weeks, I learned how to create and optimize the BMI calculation logic while working on a user-friendly design for the interface. This is part of my Portfolio Project for Holberton School. You can find the GitHub repository for this project and follow my journey as I continue to build projects and enhance my skills.

Key Features

BMI Calculation

The application calculates the BMI value based on user inputs for weight (in kilograms) and height (in meters).

BMI Categories

Once the BMI is calculated, the application categorizes the user into Underweight, Normal weight, Overweight, or Obese.
